# 🚀 AI-Powered Consumer Intelligence System

## 📌 Project Overview

This project is an AI-driven consumer analytics and product intelligence system developed using Python, NLP, Machine Learning, and Streamlit.

The system analyzes Amazon product reviews to generate:
- Customer sentiment insights
- Product intelligence
- Recommendation systems
- Strategic business analytics
- Executive dashboards

This project was developed as part of an MBA Data Science & AI analytics project.

---

# 🎯 Objectives

- Analyze customer reviews using NLP
- Detect customer sentiment using AI
- Identify hidden dissatisfaction patterns
- Build product recommendation engine
- Generate business intelligence insights
- Develop interactive analytics dashboard

---

# 🛠️ Technologies Used

- Python
- Pandas
- Plotly
- Streamlit
- Scikit-learn
- VADER Sentiment Analysis
- Machine Learning
- NLP

# 📊 Features

## ✅ AI Sentiment Analysis

## ✅ Product Intelligence

## ✅ Interactive Dashboard
- KPI metrics
- Dynamic visualizations
- Interactive filters

---

# 📈 Key Insights

- Over 90% customer reviews showed positive sentiment
- Fire Tablets dominated customer engagement
- Hidden dissatisfaction detected in highly-rated products
- Ease of use and affordability were key satisfaction drivers
